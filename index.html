Qué resuelve esta web

No es una página decorativa. Es una extensión operativa del libro. Quien llegue aquí debe poder entender la tesis, ver la portada, ir a Amazon y usar las calculadoras sin descargar nada.

  • Replica la lógica del Excel del libro.
  • Conecta los módulos entre sí, como en la hoja original.
  • Sirve como landing comercial y como herramienta pública.
  • Se puede subir gratis a GitHub Pages.

Cómo usarlo

Empieza por el módulo de gastos y ahorro. A partir de ahí, varias calculadoras heredan valores automáticamente: fondo de tranquilidad, número de la paz y asset allocation. Puedes tocar todos los inputs y ver el efecto inmediato.

Nota: la lógica se ha replicado siguiendo el comportamiento del Excel tal como está planteado en el archivo de soporte.

FAQ

Preguntas habituales

¿Se puede publicar gratis?

Sí. Esta estructura está pensada para subirla a GitHub Pages sin coste. Solo necesitas un repositorio con estos archivos.

¿Hace falta descargar el Excel?

No. La idea de esta versión es que las calculadoras estén accesibles para todo el mundo directamente desde la web.

¿Los enlaces de Amazon ya son definitivos?

No. Ahora mismo son placeholders. Sustitúyelos por la URL real de tu ficha en KDP cuando el libro esté publicado.

¿La lógica es igual a la del archivo?

Se ha replicado siguiendo el comportamiento del Excel de soporte del libro, incluyendo dependencias entre módulos.
