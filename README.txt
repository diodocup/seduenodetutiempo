Publicación gratuita con GitHub Pages

1) Crea una cuenta en GitHub.
2) Crea un repositorio nuevo, por ejemplo: seduenodetutiempo
3) Sube el contenido de esta carpeta tal cual.
4) En GitHub entra en Settings > Pages.
5) En Source elige "Deploy from a branch".
6) Selecciona la rama main y la carpeta /(root).
7) Guarda.
8) La web quedará publicada en una URL del tipo:
   https://tuusuario.github.io/seduenodetutiempo/

Antes de publicar:
- Sustituye los enlaces de Amazon de index.html por la URL real de tu libro.
- Si quieres usar otro dominio, puedes conectarlo después desde GitHub Pages.
